I learned how to put tooltips on text. This would be useful for acronyms or words that may not be clear to my audience. Simply put a title attribute on the spanned text. Hover over me!

<span>title="This is a tooltip!">Hover over me!</span>

Apparently, you can also store variables in vanilla css. Using the var() function, you can define a css value to a variable.

:root {
    --big-size: 100px;
}

.var {
    font-size: var(--big-size);
}

<p>I will make this <span class="var">word</span> very big!</p>
