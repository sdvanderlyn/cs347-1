I learned how to do an animation using css. Without javascript I can use a bunch of css attributes to specify how long the animation lasts, number of repetitions, and how the animation transitions.

animation!
@keyframes fade-anim {
    from {background-color: red;}
    to {background-color: black;}
}

#anim-block {
    padding: 20px;
    background-color: #ffffff;
    animation-name: fade-anim;
    animation-duration: 3s;
    animation-iteration-count: infinite;
    animation-direction: alternate;
}
